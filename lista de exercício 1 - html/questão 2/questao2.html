HTML Básico

HTML é a linguagem de marcação padrão para criar páginas da Web.

Modificando a Fonte

Exploraremos, agora, o comando FONT para modificarmos a cor, o tamanho e a prória
fonte do texto.
Dica:

Para que os títulos desta página ficassem numa cor diferente do texto, foram utilizados ao mesmo tempo os
comandos FONT e H1, H3 ou H4. Porém, é válido lembrar que ao misturarmos comandos de formatação lógica
com os comando de formatação física, podemos estar restringindo o efeito do comando baseado em formatação
lógica.
